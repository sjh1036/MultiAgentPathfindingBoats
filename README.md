To use:

Run ui.py, input problem specs to generate a problem, or input CSV file with each row being a survey line in the form: "x1,y1,x2,y2"

The types of problem generation are as follows: 
0: a single column instance
1: a randomly generated instance
2: a broken column instance in which a random (50)% of every survey line is missing
3: a segmented column instance in which each survey line is segmented into (5) segments
4: a broken segmented column instance in which the survey lines are segmented and only a random (10)% are used in the problem
